**Weather Watch** is an intuitive weather monitoring platform that provides real-time weather updates, hourly forecasts, historical weather data, and future forecasts. Users can also set personalized alerts for temperature thresholds. The platform is designed with a responsive user interface that supports both dark and light modes for enhanced user experience.

[Weather Watch Live](https://weather-watch-frontend.onrender.com/)

## üîë Key Features

### 1. Live Weather Tracking
Stay updated with real-time weather conditions such as temperature, wind speed, humidity, and visibility for your selected location.

### 2. Hourly Weather Updates
Receive detailed hourly weather forecasts to help you plan your day with precision.

### 3. Historical Weather Data
Explore past weather data to understand trends and make better-informed decisions.

### 4. Future Weather Forecasts
Plan ahead with future forecasts for the upcoming days, helping you prepare for various weather conditions.

### 5. Personalized Alerts
Set custom alerts for when the temperature crosses a predefined threshold, ensuring you stay updated on critical weather changes.

### 6. Dark and Light Mode
Choose between dark and light themes for a better viewing experience, reducing eye strain in low-light conditions or making the UI brighter during daytime use.

### 7. Responsive Design
Weather Watch is optimized for devices of all screen sizes, including mobile phones, tablets, and desktops, ensuring a seamless experience.

### 8. Personalized Experience
Weather Watch saves your selected location using cookies, so you don't need to re-enter your city or location each time you visit the platform.

## üõ†Ô∏è Technologies Used

### Frontend:
- **React**: A powerful JavaScript library for building user interfaces.
- **Tailwind CSS**: A utility-first CSS framework used for creating a highly customizable design with responsive elements.

### Backend:
- **Express.js**: A fast, minimalist web framework for Node.js.
- **Node.js**: A JavaScript runtime built on Chrome's V8 engine.
- **MongoDB**: A NoSQL database for storing weather data and user preferences.

### API Integration:
- **OpenWeatherMap API**: Used for fetching accurate and timely weather data.

### Deployment:
- **Render**: The platform is deployed using Render for both the frontend and backend.

## üìÇ How to Run the Project Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/AmanShukla15/weather-watch-frontend.git
   ```

2. Navigate to the project directory:
   ```bash
   cd weather-watch-frontend
   ```

3. Install the required dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to `http://localhost:3000` to view the project.
